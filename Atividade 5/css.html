CSS - Cascading Style Sheets

O que é CSS?


CSS (Cascading Style Sheets) é a linguagem usada para definir a aparência de uma página web. Enquanto o HTML cria a estrutura (títulos, textos, formulários), o CSS cuida do visual (cores, tamanhos, espaçamento, layout).


Para que serve o CSS?


  • Deixar o site mais bonito e organizado
  • Melhorar a experiência do usuário
  • Adaptar o visual para diferentes dispositivos (celular, tablet, computador)
  • Separar conteúdo (HTML) de aparência (CSS)
  • Criar animações e efeitos visuais

Estilização com CSS


Com CSS, você pode controlar praticamente todos os aspectos visuais de um site. Isso inclui:

  • Centralizar conteúdo
  • Usar Flexbox
  • Aplicar cores
/* Seleciona todas as tags 
 para cor e fonte */
h1 {
    color: #333;
    font-size: 24px;
}

/* Seleciona todas as tags 
 para cor e espaçamento */
p {
    color: blue;
    line-height: 1.5;
}

/* Seleciona a tag  para padrão do site */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}
